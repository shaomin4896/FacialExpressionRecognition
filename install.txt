软件开发环境:python3.8 + pycharm
注：由于该程序启动时需要加载模型，需要一点时间，请稍等9秒左右，界面才会显示。

环境配置步骤：
【第一步：安装python3.8】
方法一：直接在python官网下载pyhon3.8的exe文件，安装即可
方法二：
如果有ananconda，可以使用命令"conda create -n env_py python=3.8"创建3.8的虚拟环境
然后激活虚拟环境“conda activate env_py”,然后再进行第二步依赖库的安装

【第二步：安装软件所需的依赖库】
（注意：输入命令前，需先进入FacialExpressionRecognition项目目录中）
方式一:先安装requirements.txt中的第三方库，然后安装libs目录下的两个库
按照顺序运行下面3条命令：
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install libs/dlib-19.19.0-cp38-cp38-win_amd64.whl
pip install libs/face_recognition-1.3.0-py2.py3-none-any.whl

方式二：直接运行installPackages.py安装第三方库的脚本
python installPackages.py
